# ZED

A simple, leight weighed editor written in ZIG.
Currently experimental mode.

## Installation

```bash
$ git clone https://github.com/eislerMeinName/ZED.git
$ cd ZED
$ zig build
$ cd
$ cd /usr/bin
$ sudo ln -s /path/to/your/program/ZED zed
```
